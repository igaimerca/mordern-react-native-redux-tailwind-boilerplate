### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/igaimerca/mordern-react-native-redux-tailwind-boilerplate.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
2. Run in DEV mode
   ```sh
   npm start
   ```
